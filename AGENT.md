# What is this project?

Minecraftの開発環境構築・自動化を目的とした、crafters-toolboxという開発用CLIツールを作成している。

ターゲットはwindows/linuxで、`deno install`を用いてシステムに導入して利用される想定である。

ワールド・データパック・プラグイン・mod・サーバーを構成し、開発の補助と配布形式へのエクスポートを行う。

また、プラグインやmodのコンパイル->サーバーへの配置をワンコマンドで行い、サーバーも自動構築して起動するように設計しています。

`examples/`ディレクトリでは、実際に動作するプロジェクトの設定を置いています。
